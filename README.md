🎮 Neon Breaker

Neon Breaker é um jogo arcade inspirado nos clássicos Brick Breaker, desenvolvido utilizando HTML5, CSS3 e JavaScript puro. O jogador controla uma barra na parte inferior da tela e deve rebater a bola para destruir todos os blocos do cenário, acumulando pontos e tentando alcançar o maior recorde possível.

🚀 Funcionalidades
Sistema de pontuação em tempo real
Recorde salvo localmente (Local Storage)
Sistema de vidas
Tela de vitória
Tela de Game Over
Efeitos sonoros
Música de fundo
Partículas de destruição dos blocos
Fundo animado com estrelas
Interface inspirada em jogos arcade retrô
Reinício da partida utilizando a tecla ENTER
