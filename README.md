# 🎮 Neon Breaker

Neon Breaker é um jogo arcade desenvolvido com HTML5, CSS3 e JavaScript utilizando a API Canvas.

O objetivo é controlar a barra, rebater a bola e destruir todos os blocos do cenário sem perder suas vidas.

## ✨ Funcionalidades

* 🎯 Sistema de pontuação
* 🏆 Recorde salvo localmente
* ❤️ Sistema de vidas
* 🎵 Música de fundo e efeitos sonoros
* 💥 Partículas ao destruir blocos
* 🌌 Fundo animado
* 🕹️ Interface estilo arcade retrô
* 🎉 Tela de vitória
* ☠️ Tela de Game Over

## 🎮 Controles

| Tecla | Ação                      |
| ----- | ------------------------- |
| ←     | Mover para esquerda       |
| →     | Mover para direita        |
| ENTER | Iniciar/Reiniciar partida |

## 🛠️ Tecnologias

* HTML5
* CSS3
* JavaScript
* Canvas API
* Local Storage


## 🚀 Como Executar

Basta abrir o arquivo `index.html` em um navegador moderno.
