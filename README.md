🎮 Neon Breaker

Neon Breaker é um jogo arcade inspirado nos clássicos Brick Breaker, desenvolvido utilizando HTML5, CSS3 e JavaScript puro. O jogador controla uma barra na parte inferior da tela e deve rebater a bola para destruir todos os blocos do cenário, acumulando pontos e tentando alcançar o maior recorde possível.

🚀 Funcionalidades
Sistema de pontuação em tempo real
Recorde salvo localmente (Local Storage)
Sistema de vidas
Tela de vitória
Tela de Game Over
Efeitos sonoros
Música de fundo
Partículas de destruição dos blocos
Fundo animado com estrelas
Interface inspirada em jogos arcade retrô
Reinício da partida utilizando a tecla ENTER

🛠️ Tecnologias Utilizadas
HTML5
CSS3
JavaScript (ES6+)
Canvas API
Local Storage

▶️ Como Executar
Faça o download ou clone este repositório.
Abra o arquivo index.html em qualquer navegador moderno.
Pressione ENTER para iniciar o jogo.

🎯 Objetivo
Destruir todos os blocos presentes no cenário sem deixar a bola cair para fora da área de jogo. Cada bloco destruído adiciona pontos ao placar. O jogo termina quando todas as vidas são perdidas.
